## Time Complexity

It is a total time taken by an algorithm to run as a function of length of the input.

there are three case:

big(O): worst case
big(θ): Avg Case
big(Ω): Best case

worst to best: 

"$O(n!) > O(2^n) > O(n^2) > O(n \log n) > O(n) > O(\log n) > O(1)$"

## Space Complexity

It is the amount of space taken by an algorithm as a function of length of i/p;

there are two types:

1. Auxillary space complexity: Space given in input will not count
2. Total Space Complexity: Space given in input will count
